# Ferramentas de Monitoramento de Bem-Estar e Saúde Mental no Trabalho

## Status do Projeto
- **Status**: Em desenvolvimento
- **Versão**: 1.0



## Sobre o Projeto

Este projeto é uma plataforma de monitoramento de bem-estar e saúde mental voltada para o ambiente de trabalho. O sistema permite aos usuários registrar seu estado emocional, nível de estresse, energia e hábitos diários. Além disso, eles podem receber alertas sobre possíveis mudanças no estado de saúde mental, promovendo a melhoria contínua do bem-estar no ambiente corporativo.

---

## Tecnologias Utilizadas

- **Frontend**:
  - React.js
  - Vite
  - TypeScript
  - TailwindCSS
  
- **Backend**:
  - Java para criar APIs RESTful para CRUD de dados

- **Deployment**:
  - Vercel
  - API hospedada externamente
